## Basic Usage

1. Create a new note with the + button
2. Use markdown for formatting
3. Add #tags in your content
4. Pin important notes
5. Use colors to categorize

## Keyboard Shortcuts

- `Ctrl/Cmd + N`: New note
- `Ctrl/Cmd + S`: Save
- `Ctrl/Cmd + F`: Search
